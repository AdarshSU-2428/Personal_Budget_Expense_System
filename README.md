# 💰 Personal Expense Manager (Java + PostgreSQL)

A desktop-based Personal Expense Manager built using Java Swing and PostgreSQL that helps users efficiently track, manage, and analyze their finances.

## 🚀 Features

- 📊 **Dashboard**
  - Real-time summary of total budget, expenses, remaining balance, and categories

- 💼 **Budget Management**
  - Create, update, and delete monthly budgets
  - Category-wise budget allocation

- 💸 **Expense Tracking**
  - Add and manage daily expenses with category, date, and description
  - Automatic total expense calculation

- 📁 **Category Management**
  - Dynamic category creation and usage across budgets and expenses

- 📈 **Reports**
  - Monthly financial reports for better analysis

- 🕘 **Version History (Audit Trail)**
  - Tracks all changes made to budgets using PostgreSQL triggers
  - Maintains old and new values for transparency

## 🛠️ Tech Stack

- **Frontend:** Java Swing (GUI)
- **Backend:** Java (JDBC)
- **Database:** PostgreSQL
- **Architecture:** Layered (UI ↔ Service ↔ DB)

## ⚙️ Key Highlights

- ✔️ Fully dynamic dashboard (no hardcoded values)
- ✔️ Database-driven design with relational integrity
- ✔️ Trigger-based version tracking system
- ✔️ Clean and responsive UI design
- ✔️ Modular and scalable code structure

## 📌 Future Enhancements

- 📊 Graphs and analytics (charts)
- 📤 Export reports (PDF/Excel)
- 🔐 User authentication improvements
- ☁️ Cloud database integration
